# 데이터 업데이트 워크플로우

> 최종 업데이트: 2026-02-12

---

## 명령어 순서

### 1단계: 엑셀 데이터 추가
엑셀 파일을 첨부하고 아래 명령어를 입력합니다:

```
첨부 엑셀에서 [기간]의 NDA/BLA ETC 품목을 선별하여 fdaData.ts에 추가해주세요
```

- AI가 NDA/BLA 및 전문의약품(ETC)을 자동 필터링
- 치료영역, 국문 적응증, 항암제 여부 등 메타데이터 자동 할당

### 2단계: 추가/제외 목록 검토
- AI가 제시한 추가 품목 및 제외 품목 목록을 검토
- 승인 또는 수정 요청

### 3단계: 대시보드 검증
- 대시보드에 데이터가 정상 표시되는지 확인
- FDA 사이트 링크가 정상 작동하는지 검증 요청

### 4단계: 수집일 업데이트
```
대시보드의 수집일을 [오늘 날짜]로 업데이트해주세요
```

### 5단계: 클라우드 확정 및 배포
```
클라우드에 데이터를 확정(커밋)하고 라이브 사이트에 배포해주세요
```

### 6단계: 라이브 배포
- Lovable의 **Publish** 버튼을 클릭하여 라이브 사이트에 최종 반영

---

## 대상 파일

| 파일 | 용도 |
|------|------|
| `src/data/fdaData.ts` | FDA 승인 데이터 저장 |
| `src/components/dashboard/Header.tsx` | 수집일 표시 |
